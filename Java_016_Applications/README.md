# 데이터 파일에 저장하기

* 변수, 배열, 객체 에 저장되어 있는 데이터는 프로젝트가 실행되는 동안에만
	저장된 값을 유지함
	
* 변수, 배열, 객체는 주기억장칭(RAM)에 저장이 되고, 주기억장치는ㄴ 반도체 메모리로
 	휘발성 성질을 갖는다.
 	
 * 휘발성 성질을 갖는 메모리는 전원이 유지되는 동안에만 값을 저장
 
 * 데이터를 입력하여 어떤 연산을 수행할 때, 연산을 수행할 때마다 
 	데이터를 입력하여 처리한다면 매우 불편
 	
 * 입력된 데이터를 보조기억장치(HDD, SDD, USB) 에 저장해 두면
 	프로젝트가 종료되거나 컴퓨터가 종료되어도 그 데이터를 보존가능
 
 * 보조기억장치(storage) 에 데이터를 저장하는 것 == 영속성 유지
 
 * 영속성 유지란 한 번 저장된 데이터는 언제든지 다시 재생하여 사용하는 것
 
 * 가장 쉬운 방법은 파일에 저장 / 가장 좋은 방법은 DataBase에 저장
 
 

## 컴퓨터 파일
Text 파일과 BIN(BInary, 이진) 파일 존재

Text 파일 : 윈도우의 메모장 , sublime, Text 와 같은 도구를 사용하여 파일을 열었을때
			내용을 어려움 없이 읽을 수 있는 파일
 			: 내용이 ASCII코드값으로 변환되어 저장		
	  
			
BIN 파일 : 이미지, 사진, 동영상, 음악, HWL, XLS, PPT 등의 파일들로 메모장으로 열었을때 
			내용이 깨짐 현상이 있어 정확히 읽을 수  없는 파일
		 : 파일마다 독특한 방법으로 저장이 되고 모든 내용이 이진수들의 나열로 저장
		
FileWriter, PrintWriter 는 Text파일을 만들고 Text방식으로 저장하는데 사용하는 클래스

FileReader, BufferedReader는 Text 파일을 읽어서 컴퓨터메모리로 가져올때 사용하는 클래스
